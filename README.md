# 世界杯完美阵容挑战 🏆

原为虎扑APP内的互动活动，这是修改后的网页版，**无需虎扑APP**，在浏览器中直接打开即可游玩。

## 修改内容

- 每个位置的随机选人次数改为无限制
- 移除虎扑APP检测，模拟比赛、生成海报等功能直接在网页端进行
- 添加了兼容补丁，独立 HTML 文件可随处托管

## 玩法

1. 选择阵型（4-3-3、4-2-3-1 等）
2. 点击每个位置随机抽取球员（可无限次重抽）
3. 完成阵容后点击"开始模拟8场"
4. 查看晋级结果，生成海报分享

## 技术说明

- 单文件 HTML，所有逻辑内嵌
- 球员头像、字体等资源从虎扑 CDN 加载
- 比分提交到虎扑后端的接口可能受限，不影响核心玩法
